Représentation d’un signal¶

Représentez entre 0 et 10 s la sinusoïde amortie

\[\begin{split} x(t) = \begin{cases} 0 \text{ si } t<0 \\ A e^{-at} \sin(2 \pi f t + \varphi) \text{ si } t\geq 0 \\ \end{cases} \end{split}\]

où \(A=2\), \(a=0,5\), \(\varphi=1\) et \(f = 2\) Hz. Choisissez un pas de temps correct pour obtenir une jolie courbe.

Produit de convolution¶

Représentez le produit de convolution \((x*y)[n]\) où :

\[\begin{split} x[n] = \begin{cases} +1 \text{ pour } n=1, \\ -1 \text{ pour } n=-1, \\ 0 \text{ ailleurs}, \end{cases} \qquad\qquad y[n] = \begin{cases} 2 &\text{ pour } |n| \leq 1, \\ 0 &\text{ ailleurs}. \end{cases} \end{split}\]

Pour définir \(x[n]\), remarquez que le signal correspond à la somme de deux impulsions discrètes. Et pour définir une impulsion discrète, vous pouvez utiliser la fonction numpy.where (voir sa syntaxe). Par exemple, l’impulsion

\[\begin{split} z[n] = \begin{cases} A &\text{ pour } n = m, \\ 0 &\text{ ailleurs}. \end{cases} \end{split}\]

peut être définie par :

z = np.where( n=m, A, 0)

Transformée de Fourier discrète¶

On considère le signal créneau suivant de période \(N=100\) :

\[\begin{split} x[n] = \begin{cases} 1 \quad\text{ si } 0 \leq n < 50, \\ 0 \quad\text{ si } 50 \leq n < 100. \end{cases} \end{split}\]

  • Créez et affichez le signal \(x[n]\) sur 100 échantillons.

  • Tracez la transformée de Fourier discrète (TFD) \(X[k]\) de \(x[n]\) entre \(0\) et \(N-1\) (scipy.fft.fft).

  • Centrez la TFD pour la représenter entre \(-N/2\) et \(N/2-1\) (scipy.fft.fftshift).

  • Modifiez les abscisses pour représenter la TFD en fonction des fréquences (et non des échantillons). On suppose que \(T_e=0,5\) s.

Analyse spectrale d’un signal audio¶

  • Chargez le signal chord.wav (wavfile.read). C’est un signal sonore enregistré en stéréo : comment le convertir en mono ? Affichez-le.

  • Affichez la TFD du signal pour déterminer les harmoniques principales.

  • Quel est l’accord joué ? Vous pouvez également l’écouter si vous avez l’oreille musicale…
